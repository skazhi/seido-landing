## 2. Предмет Соглашения

2.1. Seido — это платформа для сбора, хранения и отображения спортивных результатов бегунов.

2.2. Основные функции Сервиса:
- Регистрация и ведение профиля бегуна
- Поиск и просмотр результатов на забегах
- Сравнение результатов с другими бегунами
- Календарь предстоящих забегов
- Статистика и рейтинги

2.3. Сервис предоставляется **на условиях «как есть» (as is)**. Администрация не гарантирует бесперебойную работу и отсутствие ошибок.

## 3. Обработка персональных данных

3.1. Регистрируясь в Сервисе, Пользователь даёт согласие на обработку следующих персональных данных:
- Фамилия, имя, отчество
- Дата рождения
- Пол
- Город и страна проживания
- Спортивные результаты (время, место, дистанция)
- Telegram ID (для идентификации)

3.2. Цели обработки персональных данных:
- Идентификация Пользователя в Сервисе
- Сохранение и отображение спортивных результатов
- Формирование статистики и рейтингов
- Уведомление о новых функциях и событиях

3.3. Правовое основание обработки:
- Согласие Пользователя (ст. 6, 9 Федерального закона № 152-ФЗ «О персональных данных»)

3.4. Срок действия согласия:
- Согласие действует бессрочно до момента его отзыва Пользователем

3.5. Отзыв согласия:
- Пользователь может отозвать согласие на обработку персональных данных, направив запрос в службу поддержки
- После отзыва данные будут удалены в течение 30 дней

3.6. Передача данных третьим лицам:
- Персональные данные не передаются третьим лицам, за исключением случаев, предусмотренных законодательством РФ
- Спортивные результаты могут отображаться в общем доступе для других пользователей Сервиса

3.7. Защита данных:
- Администрация принимает технические и организационные меры для защиты персональных данных от неправомерного доступа
- Данные хранятся на серверах, расположенных на территории РФ

## 9. Использование публичных протоколов забегов

9.1. Сервис может использовать данные из публично доступных протоколов забегов, опубликованных организаторами на своих официальных сайтах.

9.2. Данные из протоколов обрабатываются в следующих целях:
- Поиск результатов участников по фамилии
- Формирование статистики и рейтингов
- Сравнение результатов между бегунами
- Отображение истории участия в забегах

9.3. Правовое основание обработки данных из протоколов:
- Публично доступные данные (ст. 6 п. 11 152-ФЗ)
- Согласие участника при регистрации в Сервисе
- Информация о результатах является фактической и не охраняется авторским правом (ст. 1259 ГК РФ)

9.4. Права участника забега:
- Участник имеет право запросить удаление своих данных из протокола
- Участник может отозвать согласие на обработку данных
- Удаление данных производится в течение 30 дней с момента запроса

9.5. Права организатора забега:
- Организатор имеет право запросить удаление данных о своём забеге
- По запросу организатора все данные о забеге будут удалены в течение 30 дней
- Сервис прекратит использование протокола данного организатора

9.6. Источники данных:
- В карточке каждого забега указывается источник данных (организатор)
- При наличии ссылки на протокол, она указывается в карточке забега
- Сервис не претендует на авторство протоколов

9.7. Ограничения:
- Сервис не является официальным источником результатов
- Официальные протоколы организаторов имеют приоритет
- При расхождении данных приоритет отдаётся официальным источникам

9.8. Контакты для запросов:
- Участники: команда `/delete` в боте или обращение в поддержку
- Организаторы: Telegram @Skazhi или email (указать при создании)

# Seido: контакты для пользователей (Шаг 3)

## Что нужно по 152-ФЗ и практике

Для запросов на удаление данных, претензий и вопросов по персональным данным нужен **доступный контакт**. Закон не указывает конкретный формат.

| Формат | Плюсы | Минусы |
|--------|-------|--------|
| **Email** | Официально, есть письменная переписка | Меньше людей пишут на почту |
| **Telegram @ник** | Удобно, много людей в TG | Менее «официально» |
| **Телефон** | Прямой контакт | Лишние звонки, приватность |

## Рекомендация

**Достаточно:** один из вариантов:
- **Email** (у вас: seido.community@yandex.ru)
- **Telegram @Skazhi** (у тебя уже есть)

Можно указать оба: «По вопросам персональных данных: email или @Skazhi в Telegram».

Где публиковать:
- В оферте (`docs/offer.md`)
- В разделе «Помощь» бота
- В политике конфиденциальности

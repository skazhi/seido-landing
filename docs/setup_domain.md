# Инструкция: Регистрация домена Seido

## Варианты доменов

### Приоритет 1 (для России)
- **seido.ru** — ~200-500 руб/год
- Регистраторы: Reg.ru, Timeweb, REG.RU

### Приоритет 2 (международные)
- **seido.app** — ~$15-20/год
- **seido.io** — ~$30-40/год
- Регистраторы: Namecheap, GoDaddy, Cloudflare

### Приоритет 3 (бюджетные)
- **seido.xyz** — ~$5-10/год
- **seido.space** — ~$10-15/год
- Регистраторы: Namecheap, Porkbun

## Шаги регистрации

### Вариант A: Reg.ru (для .ru домена)

1. **Проверить доступность**
   - Открой https://www.reg.ru/
   - Введи "seido.ru" в поиск
   - Проверь, свободен ли домен

2. **Зарегистрировать**
   - Нажми "Зарегистрировать"
   - Выбери срок (1 год минимум)
   - Заполни данные (ФИО, email, телефон)
   - Оплати (~200-500 руб)

3. **Настроить DNS**
   - После регистрации открой панель управления
   - Перейди в "DNS-серверы"
   - Если используешь GitHub Pages: укажи A-записи (см. инструкцию GitHub)
   - Если используешь Vercel: укажи CNAME на vercel-dns.com

### Вариант B: Namecheap (для .app/.io/.xyz)

1. **Проверить доступность**
   - Открой https://www.namecheap.com/
   - Введи домен в поиск
   - Проверь цену и доступность

2. **Зарегистрировать**
   - Добавь в корзину
   - Выбери срок (1 год)
   - Создай аккаунт (если нет)
   - Оплати картой

3. **Настроить DNS**
   - В панели управления доменом
   - Advanced DNS → Add New Record
   - Для GitHub Pages: A-записи (185.199.108.153, 185.199.109.153, 185.199.110.153, 185.199.111.153)
   - Для Vercel: CNAME на cname.vercel-dns.com

### Вариант C: Cloudflare (рекомендуется)

1. **Зарегистрировать домен**
   - Можно через Cloudflare Registrar (часто дешевле)
   - Или купить на другом регистраторе и перенести

2. **Настроить DNS**
   - Добавь домен в Cloudflare
   - Настрой DNS-записи
   - Cloudflare даст бесплатный SSL и CDN

## Настройка для GitHub Pages

1. В репозитории GitHub:
   - Settings → Pages
   - Custom domain → введи "seido.ru" (или другой)
   - GitHub покажет инструкции по DNS

2. В панели регистратора:
   - Добавь A-записи (IP-адреса от GitHub)
   - Или CNAME на `твой-ник.github.io`

## Настройка для Vercel

1. В проекте Vercel:
   - Settings → Domains
   - Add domain → введи "seido.ru"
   - Vercel покажет DNS-записи

2. В панели регистратора:
   - Добавь CNAME на `cname.vercel-dns.com`
   - Или A-записи (IP от Vercel)

## Проверка

После настройки DNS подожди 5-60 минут (проверка DNS) и проверь:
```bash
# В терминале
nslookup seido.ru
# или
dig seido.ru
```

Должны вернуться правильные IP-адреса.

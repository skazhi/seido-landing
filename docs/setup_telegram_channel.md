# Инструкция: Создание Telegram-канала Seido

## Шаги

### 1. Создать канал
✅ **Канал уже создан:** @seido_community

Если нужно создать новый канал:
1. Открой Telegram
2. Нажми на меню (три полоски) → "Новый канал"
3. Введи название: **Seido — Беговая статистика**
4. Добавь описание: "Единая платформа для бегунов. Все результаты, все старты, всё сообщество в одном месте."
5. Выбери тип: **Публичный канал**
6. Придумай юзернейм: `@seido_community`
7. Нажми "Создать"

### 2. Настроить канал
✅ **Канал настроен:** @seido_community
- Лого загружено
- Описание с ссылкой на бота добавлено
- Приветственный пост опубликован

**Ссылка на канал:** https://t.me/seido_community

### 3. Опубликовать первый пост
1. Открой канал
2. Скопируй текст из `telegram_content.md` (раздел "Приветственный пост")
3. Вставь в канал и опубликуй
4. Закрепи пост (долгое нажатие → "Закрепить")

### 4. Создать чат (опционально)
1. В настройках канала → "Обсуждение"
2. Создать группу для обсуждений
3. Или создать отдельный чат: "Новая группа" → "Seido Chat"

### 5. Пригласить первых подписчиков
1. Отправь ссылку на канал друзьям-бегунам
2. Попроси подписаться и поделиться
3. Цель: минимум 5 подписчиков для начала

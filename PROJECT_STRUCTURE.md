# Seido — Структура проекта

```
SayDo/
│
├── SayDo_plan.md           # Полный план развития проекта
├── README.md               # Описание проекта
├── QUICKSTART.md           # Быстрый старт (инструкция по запуску)
├── index.html              # Лендинг-заглушка
├── telegram_content.md     # Контент для Telegram (посты, контент-план)
│
├── bot/                    # Telegram-бот
│   ├── main.py             # Точка входа
│   ├── db.py               # Работа с базой данных
│   ├── handlers.py         # Обработчики команд
│   ├── config.py           # Конфигурация
│   ├── requirements.txt    # Python-зависимости
│   ├── schema.sql          # Схема базы данных PostgreSQL
│   ├── README.md           # Документация по боту
│   └── .env.example        # Пример переменных окружения
│
├── web/                    # Веб-интерфейс (будет позже)
│   └── ...
│
└── docs/                   # Документация
    └── ...
```

## Файлы

### Основные
| Файл | Описание |
|------|----------|
| `SayDo_plan.md` | Детальный план развития по этапам |
| `README.md` | Описание проекта, миссия, команда |
| `QUICKSTART.md` | Пошаговая инструкция по запуску |
| `index.html` | Лендинг для сбора email |

### Telegram-бот
| Файл | Описание |
|------|----------|
| `bot/main.py` | Точка входа, запуск polling |
| `bot/db.py` | Все функции для работы с БД |
| `bot/handlers.py` | Обработчики команд бота |
| `bot/config.py` | Конфигурация (токен, БД) |
| `bot/schema.sql` | SQL-схема базы данных |
| `bot/requirements.txt` | Python-библиотеки |

### Контент
| Файл | Описание |
|------|----------|
| `telegram_content.md` | Посты для Telegram-канала и чата |

---

## Следующие шаги

1. **Создать Telegram-канал** → использовать тексты из `telegram_content.md`
2. **Запустить лендинг** → `index.html` на GitHub Pages / Vercel
3. **Настроить бота** → следовать `bot/README.md` и `QUICKSTART.md`
4. **Наполнить базу** → добавить первые забеги и результаты

## Технологии

- **Бот:** Python 3.10+, aiogram 3.x, asyncpg, SQLAlchemy
- **База данных:** PostgreSQL 14+
- **Хостинг:** Vercel / GitHub Pages (лендинг), VPS / Render (бот + БД)
- **Платежи (позже):** ЮKassa / Тинькофф
